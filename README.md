# Root Source
This is an open source web server to promote and find other open source projects. This is nothing groundbreaking -- something git hub does already but a fun practice of go and portfolio piece.


## Funcionality of the site
 - login (oauth)
 - link a github repo
   - parse repo for tags, languages and other contributors
 - look for repos
 - suggest repos based on other projects you have worked on
 - chat about a repo


## Technology
 - Go backend
 - Postges DB
 - React => webcomponents
